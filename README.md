# 🚀 Overview

AuraOS is an **AI Operating Layer** built on top of existing operating systems that transforms the way humans interact with computers.

Instead of relying on a mouse, keyboard, or trackpad, AuraOS enables users to interact naturally using:

- ✋ Hand Gestures
- 🎤 Voice Commands
- 🧠 Context-Aware AI
- 👀 Computer Vision
- ⚡ Intelligent Workflow Automation

AuraOS is **not another AI assistant.**

It is an operating layer that understands your intent and seamlessly interacts with your laptop.
